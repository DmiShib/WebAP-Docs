대시보드 (Dashboard) ​

Dashboard는 강력한 게임 내 제어 패널이자 프로파일러입니다. 플러그인의 기능을 모니터링하고, 프레임 시간을 분석하며, 실행 중인 빌드에서 직접 Scaler 레벨을 수동으로 재정의할 수 있습니다.

사용 방법 ​

프로젝트의 아무 씬에나 Dashboard.prefab을 드래그 앤 드롭하세요.
이 프리팹은 자동으로 DontDestroyOnLoad로 설정되며 씬 전환 시에도 활성 상태를 유지합니다.

프로덕션 빌드 참고 사항

대시보드는 플러그인의 핵심(core)과 완전히 분리되어 있습니다. 게임을 출시하기 전에 씬에서 이 프리팹을 삭제하기만 하면 됩니다. Unity의 Code Stripping 시스템이 최종 빌드에서 대시보드의 모든 코드를 자동으로 제거하여(Resources 폴더 팽창 방지), 프로젝트 용량을 최소한으로 유지합니다. WebAP 코어는 백그라운드에서 눈에 띄지 않게 계속 실행됩니다.

왼쪽 패널 (제어) ​

대시보드의 왼쪽은 플러그인과 실시간으로 상호 작용하기 위해 설계되었습니다.

  • Enable Web AP
    플러그인 코어의 전역 스위치입니다. 비활성화하면 자동 최적화가 중지되고 모든 그래픽 매개변수가 원래의(최대) 값으로 되돌아갑니다.
  • Anti Yo-Yo Protection
    페널티 록아웃(Penalty Lockout)을 켜거나 끕니다. 비활성화된 경우 플러그인은 일시적인 차단 없이 그래픽 품질을 지속적으로 높이고 낮출 수 있습니다.
    (상세 내용: 안티 요요 시스템)
  • Active Scalers
    시스템에 추가된 모든 최적화 모듈 목록입니다.
    • Toggle (토글): 특정 Scaler를 동적으로 완전히 활성화하거나 비활성화합니다.
    • Slider (슬라이더): 품질 레벨을 수동으로 무시합니다(Override Level). 0은 최대 품질을 의미합니다. 슬라이더를 움직이면 Indexer가 이 Scaler의 관리를 중지하므로 게임에 미치는 영향을 시각적으로 평가할 수 있습니다.
    • Resolution Mode (해상도 모드): Resolution Scaler에만 있는 설정입니다(BiRP 모드). NativeBrowserScaling 전략과 RenderTextureScaling 전략 사이를 동적으로 전환할 수 있습니다.
      (상세 내용: Scaler 레퍼런스)

오른쪽 패널 (분석) ​

오른쪽은 엔진의 원시 지표와 플러그인 "두뇌"의 내부 논리를 표시합니다. 차트를 부드럽게 만들기 위해 Zero-GC 지수 이동 평균(Exponential Moving Average) 알고리즘이 사용됩니다.

지표 및 타이밍 (Metrics & Timings) ​

  • Target Min FPS: 프로젝트 설정에서 가져온 기본 성능 목표입니다.
  • Current FPS: 현재 프레임 속도와 과거 스파이크 그래프입니다.
  • Current Scale / DPI: 현재 해상도 승수입니다. 1.00은 네이티브 해상도를 나타냅니다. 더 낮은 값(예: 0.50)은 GPU를 절약하기 위해 3D 세계의 해상도가 축소되었음을 나타냅니다.
  • Frame Time / CPU Time: FrameTimingManager를 통해 수집된 정확한 하드웨어 시간입니다. 프로세서가 프레임 로직에 실제 시간을 얼마나 소비하는지 이해하는 데 도움이 됩니다.

플러그인 두뇌 (Plugin Brain) ​

결정 파이프라인의 논리를 표시합니다.
(상세 내용: 작동 방식)

  • Bottleneck: 현재 지연을 유발하는 디바이스 하위 시스템이 무엇인지 식별합니다(CPU, GPU, TargetFrameRate 또는 Unknown).
  • Action: Indexer가 무엇을 하려는지 나타냅니다(IncreaseQuality, DecreaseQuality 또는 Stale - 조치가 필요 없는 경우).
  • Status (Action 아래): 시스템 상태입니다: Monitoring...(프레임 평가 중), Cooldown(그래픽 변경 후 안정화 대기 중) 또는 Penalty Lockout(안티 요요 페널티 활성 상태).

활성화 됨 (Active) ​

  • 성능을 보존하기 위해 시스템에서 현재 레벨이 낮아진 Scaler 목록입니다. FPS가 높고 모든 것이 최적화된 경우 - None으로 표시됩니다.
